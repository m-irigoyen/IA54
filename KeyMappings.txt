	SIMULATOR WINDOW
F1 : load problem rocket terrain editor
F2 : load problem rocket (one thruster)
F3 : load problem rocket2 (two thrusters)

Esc (or close window) : exit program
O : toggle wave optimised drawing
W : toggle display waves
E : toggle display emitters
R : toggle display receptors
S : toggle display simulator (equivalent to toggling the 3 above)

Mouse : drag & drop bodies


	PROBLEM WINDOW
ProblemRocket :
F1 : Activate manual controls
F2 : deactivate manual controls
Arrow keys : control rocket angle and engine power
P : Play/Pause
A : Toggle relative angles
O/I : Increment/Decrement problem Speed
S : Save current terrain
L : Load current terrain

Rocket One engine :
Up/Down : Control engine power
Left/Right : Control rocket angle

Rocket Two Engines : 
A/Q : Control left engine power
Z/S : Control left engine power
Left/Right : Control rocket angle (easier to test with that
Up/Down : Control engine power on both engines at the same time

Rocket Terrain editor :
S/L : save/load
B : Load base terrain
D : Load default terrain
W : Set terrain wind
R : Set rocket starting position
V : set rocket starting velocity
